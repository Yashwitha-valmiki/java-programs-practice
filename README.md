## ▶️ How to Run Programs

1. Clone the repository:
   ```bash
   git clone https://github.com/Yashwitha-valmiki/java-programs-practice.git
   ```
2. Move into the project directory:
   ```bash
   cd java-programs-practice
   ```
3. Compile a Java file:
   ```bash
   javac FileName.java
   ```
4. Run the program:
   ```bash
   java FileName
   ```
